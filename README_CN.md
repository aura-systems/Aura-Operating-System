## 支持的特性

请看 [Aura进程表](https://github.com/aura-systems/Aura-Operating-System/projects/4) 或者我们的 [路线图](https://github.com/aura-systems/Aura-Operating-System/projects/3) 来了解我们之后将会推出什么。

* ATA IDE / AHCI 驱动器。
* FAT32/16/12 + VFS系统。
* PCI 设备扫描。
* PS/2 Keyboard + Mouse.
* 重新启动 + ACPI 关机。
* 多语言。
* 命令解释器
* 错误捕捉。
* GUI (SVGAII, VBE)
* 网络 (ARP, IPV4, ICMP, UDP, DNS, DHCP, TCP, HTTP)
